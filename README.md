<h2>📌 Project Overview</h2>

<p>
The <b>Car Price Predictor</b> is a Machine Learning based web application that estimates the price of a used car using features such as:
</p>

<ul>
<li>Car Company</li>
<li>Car Model</li>
<li>Year of Purchase</li>
<li>Fuel Type</li>
<li>Kilometers Driven</li>
</ul>

<p>
The system uses a trained <b>Linear Regression Machine Learning model</b> to predict the price of the car based on historical data.
</p>

<h2>📊 Machine Learning Workflow</h2>

<ol>
<li>Collect Car Dataset</li>
<li>Clean and preprocess the data</li>
<li>Perform data analysis</li>
<li>Train RandomForestRegressor</li>
<li>Save trained model (.pkl file)</li>
<li>Predict price using user input</li>
</ol>

<h2>🚀 Installation & Setup</h2>

<h3>1️⃣ Clone the Repository</h3>

<pre>
git clone https://github.com/PrajapatiPramodPandit/Car-Price-Predictor-application.git
</pre>

<h3>2️⃣ Navigate to Project Folder</h3>

<pre>
cd Car-Price-Predictor-application
</pre>

<h3>3️⃣ Install Dependencies</h3>

<pre>
pip install -r requirements.txt
</pre>

<h3>4️⃣ Run the Application</h3>

<pre>
python application.py
</pre>
